AeroRouteMap Privacy Policy

Last updated: May 25, 2026

AeroRouteMap is an iOS flight-planning reference and route visualization app. This policy describes the current App Store build.

Data collection

AeroRouteMap does not require an account, does not request device location permission, and does not include advertising, tracking, analytics, or crash-reporting SDKs. The developer does not collect personal data from the app.

Route and waypoint input

Route text, custom waypoints, coordinate shapes, takeoff time, and flight time values are processed on device. AeroRouteMap does not upload those inputs to the developer or to a route-processing server.

NavData updates

When you tap the NavData update button, AeroRouteMap downloads a public manifest and data package from GitHub Pages, validates the package checksum, and stores the installed CSV locally on your device. The update request does not include your entered routes or custom waypoints.

GitHub Pages may receive standard web request information such as IP address and user-agent as part of delivering the static files. AeroRouteMap does not control GitHub's infrastructure logs.

Local storage

Installed NavData packages and metadata are stored locally in the app's Application Support container. Deleting the app removes this local app data from the device.

Operational limitation

AeroRouteMap data is for planning and reference only. It is not certified navigation data and must not be used for operational navigation, dispatch, air traffic control, terrain clearance, or flight safety decisions.

Contact

For privacy questions, use the AeroRouteMap support page at https://fox3nova.github.io/aeroroutemap/support.html or open a GitHub issue at https://github.com/fox3nova/aeroroutemap/issues.
